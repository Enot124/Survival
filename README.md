# Survival
 3d first person shooter. The goal of the game is to survive the maximum amount of time and kill the largest number of enemies. There are two locations available in the game.<br>

Control:<br>
WASD - movement<br>
R - reload<br>
Spacebar - jump<br>
esc - menu.
